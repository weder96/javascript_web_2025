# Extract method/function

You have a code fragment that can be grouped together.

```javascript
var posts = getPosts();
posts.forEach(function (post) {
  console.log(post);
});
```

Extract the fragment into a new function/method.

```javascript
var logPosts = function (posts) {
  posts.forEach(function (post) {
    console.log(post);
  });
};

var posts = getPosts();
logPosts(posts);
```
